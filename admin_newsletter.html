🔗 Brevo (דיוור)
בודק חיבור...
✅ נרשמים חדשים מתווספים ל-Brevo אוטומטית בזמן אמת.
הכפתור לעיל מסנכרן את כל הרשימה הקיימת.
🚀 הגדרת Brevo
1
פתח חשבון חינמי ב-brevo.com
(חינמי עד 300 מיילים/יום, רשימה ללא הגבלה)
2
לך ל: Settings → API Keys → Create API Key
העתק את המפתח
3
ב-Railway → Variables הוסף:
BREVO_API_KEY = your-key-here
BREVO_LIST_ID = 2
(ה-List ID מופיע ב-Brevo → Contacts → Lists)
4
לחץ "סנכרן עכשיו" — כל הרשימה תועלה.
מכאן כל נרשם חדש יתווסף אוטומטית.
5
לשליחת ניוזלטר: ב-Brevo → Email Campaigns → Create
בחר את הרשימה → עצב → שלח 🎉
